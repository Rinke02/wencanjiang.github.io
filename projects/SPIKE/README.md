# SPIKE Project Site

This is a static project page generated from the current NeurIPS_2026 paper sources. The layout is inspired by IVEBench-style benchmark project pages and Nerfies-style paper pages.

## Local Preview

Open `index.html` directly in a browser.

## Deploy

For a project URL such as:

```text
https://<username>.github.io/projects/SPIKE/
```

copy this `project-site/` directory to:

```text
<username>.github.io/projects/SPIKE/
```

and enable GitHub Pages for the repository.

## Placeholders

- arXiv link
- Code link
- Data link
- Demo link
- Public author names and affiliations
- Camera-ready BibTeX
