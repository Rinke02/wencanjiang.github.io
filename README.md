# wencanjiang.github.io

Personal website for Wencan Jiang.

This site is based on Academic Pages and is hosted with GitHub Pages.

Common content locations:

- `_config.yml`: site title, author profile, links, and global settings
- `_pages/about.md`: homepage content
- `_pages/cv.md`: CV page
- `_publications/`: publication entries
- `projects/SPIKE/`: SPIKE project page

Local preview:

```bash
bundle exec jekyll serve
```
